# Obsidian Wallabag Plugin

## Sorry, abandoning

I originally picked up development of this plugin to add a couple of improvements for I think folder validation. After the original developer ceased activity I was asked by a couple of people in the community to pick it up. I did so and made a few other changes. It was never meant to be anything more. Now, after multiple attempts to get the plugin official registered, and requests coming in for further changes I need to pull the pin now. I am unable to commit to this project any further. I don't even use it myself, preferring the [Obsidian Web Clipper](https://obsidian.md/clipper) for article capture without the intervening need for Wallabag.

## Purpose 

This plugin for [Obsidian](https://obsidian.md) allows you to sync [Wallabag](https://www.wallabag.it/en) items into Obsidian notes in various ways. It works with the official [hosted Wallabag environment](https://wallabag.it/), or your self-hosted version. The plugin is hosted at https://github.com/quantumgardener/obsidian-wallabag.

## Documentation

The plugin is fully documented in [the plugin wiki](https://github.com/quantumgardener/obsidian-wallabag/wiki).
